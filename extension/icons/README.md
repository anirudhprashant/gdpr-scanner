# Chrome Extension Icons

Icons need to be created manually (ImageMagick not available on server).

Required sizes:
- 16x16 pixels - icon16.png
- 48x48 pixels - icon48.png
- 128x128 pixels - icon128.png

Design:
- Background: White (#ffffff)
- Color: #2563eb (blue)
- Symbol: ðŸ”’ (lock/shield icon for security/compliance)

You can:
1. Create icons using online tools like: https://www.favicon-generator.org/
2. Or use a design tool like Figma and export PNGs
3. Place the 3 PNG files in this `icons/` directory

Once icons are added, the extension will display correctly in the Chrome Web Store.
